## Installation

1. Clone the repository

```
git clone https://github.com/fsid908/Weather-App.git
```

2. Navigate to the project directory

```
cd Weather-App
```

3. Add API Key

```
const apiKey = "YOUR_API_KEY";
```

4. Open `index.html` in your browser.

<li>WeatherAPI</li>
